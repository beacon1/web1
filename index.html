The HTML < link > Element
The element is used to link to external style sheets:
< link rel="stylesheet" href="mystyles.css" >
With an external style sheet, you can change the look of an entire web site, by changing one file!

This is designing the colours like above HTML in the HEAD tagname so your design style is seperate from your main HTML

An internal CSS has defined the colours, in the < head > section of this HTML page, within the < style > element.

This is an inline CSS

An external style sheet can be written in any text editor. The file must not contain any HTML code, and must be saved with a .css extension.
Here is how the "styles.css" looks (see test:html for exact setup):
body { background-color: powderblue; } h1 { color: blue; font-family: courier; font-size: 300%; } p { color: red;font-family: courier; font-size: 160%; }

you can change font size, colour & family. Alter colours by Class or by ID. Add Borders, padding & margins all in your Cascading Style Sheet.
